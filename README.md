# cabala-cli

个人开发的基础脚手架工具，提供一些方便自己开发的项目模板

# 安装

Windos系统安装

```shell
$ npm install @hanserena/cabala-cli -g
```

# 查看包信息

```shell
$ cabala
```

# 查看可用模板类型

```shell
$ cabala type
```

# 创建项目

```shell
# cabala new <type> <project-name>
$ cabala new vue test-project
$ cd ./test-project/
$ npm install
```
